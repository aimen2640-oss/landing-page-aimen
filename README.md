## ✨ Key Features
- **Modern UI:** Clean aesthetic with a professional color palette.
- **Glassmorphism:** Used for features cards to give a premium look.
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop views.
- **Interactive Elements:** Smooth hover effects on buttons and cards.
